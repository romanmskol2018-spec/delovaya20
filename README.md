# Деловая20 — Urban Light Industrial

Промо-лендинг проекта коммерческой недвижимости формата Light Industrial внутри Москвы.
Статический сайт без фреймворков и сборки: открывается двойным кликом по `index.html`.

## Структура

| Файл | Назначение |
|------|------------|
| `index.html` | Одностраничный лендинг (все секции) |
| `styles.css` | Все стили. Токены темы — в `:root` (легко заменить на фирменные) |
| `script.js` | Маска телефона, валидация, анимации, режим отправки формы |
| `send.php` | PHP-обработчик формы (отправка заявки на info@emuna.ru) |
| `privacy.html` | Политика конфиденциальности |
| `favicon.svg` | Иконка сайта |

## Локальный запуск

Просто откройте `index.html` в браузере. Либо поднимите статический сервер:

```bash
python -m http.server 8000
# затем откройте http://127.0.0.1:8000/
```

## Форма заявки: демо и боевой режим

В начале `script.js`:

```js
const DEMO_MODE = true; // демо: показывает «Спасибо» без сервера
```

- **`DEMO_MODE = true`** (по умолчанию) — форма проходит валидацию и показывает
  сообщение об успехе. Ничего никуда не отправляется. Удобно для показа и для
  GitHub Pages (там PHP не работает).
- **`DEMO_MODE = false`** — форма отправляет данные на `send.php`, который шлёт
  письмо на `info@emuna.ru`. Требуется хостинг с поддержкой PHP (`mail()`).

## Деплой на GitHub Pages

GitHub Pages отдаёт только статику, поэтому форма работает в демо-режиме
(`DEMO_MODE = true`). Для реальной отправки заявок разместите сайт на хостинге с PHP
и переключите `DEMO_MODE = false`.

## Замена контента

- Фирменные цвета — переменные в `:root` файла `styles.css`.
- Места под реальные фото помечены комментариями `<!-- MEDIA: ... -->` в `index.html`.
- Домен в тегах `canonical` / OpenGraph укажите свой после публикации.
